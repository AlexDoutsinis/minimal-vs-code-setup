- **Prettier:**

  Install Prettier globally as node package: `npm install -g prettier`.

  [Install Prettier as VS Code extension](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)

  Configuration:

  ```
  // .prettierrc

  {
    "semi": false,
    "trailingComma": "all",
    "singleQuote": true,
    "printWidth": 70,
  }
  ```
